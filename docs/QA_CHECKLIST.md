# QA 체크리스트

이 문서는 기능 구현 후 사람이 직접 확인하거나 Codex가 로직 검사로 확인해야 할 항목입니다.

## 공통

### 문법 검사

명령:

```powershell
node -e "const fs=require('fs');const html=fs.readFileSync('index.html','utf8');const scripts=[...html.matchAll(/<script[^>]*>([\s\S]*?)<\/script>/gi)].map(m=>m[1]); for (const s of scripts) new Function(s); console.log('script parse ok:', scripts.length);"
```

예상 결과:

```text
script parse ok: 1
```

### 공백 검사

명령:

```powershell
git diff --check -- index.html docs
```

예상 결과:

- 오류 없음
- Windows 줄바꿈 경고는 허용 가능

## 홈 화면

확인 항목:

- 현재 플레이어가 표시된다.
- 프로필 선택 드롭다운이 있다.
- 새 이름을 입력하고 추가하면 새 프로필이 선택된다.
- 프로필을 바꾸면 현재 플레이어 이름이 바뀐다.
- 기록 보기 버튼이 있다.
- `미로찾기` 카드가 보인다.
- `사진 퍼즐` 카드가 보인다.
- `숫자 패턴 맞추기` 카드가 깨지지 않는다.
- 각 카드 클릭 시 해당 게임 화면으로 이동한다.

## 로컬 기록

확인 항목:

- 첫 실행 시 `게스트` 유저가 자동 생성된다.
- 브라우저 localStorage에 `miniGamePlayRecords.v1` 키가 생성된다.
- 미로를 완료하면 현재 유저 기록에 추가된다.
- 사진/동영상 퍼즐을 완료하면 현재 유저 기록에 추가된다.
- 기록 보기 화면에는 현재 유저 기록만 보인다.
- 유저를 바꾸면 기록 목록도 바뀐다.

예상 저장 구조:

```js
{
  users: [],
  activeUserId: "",
  records: []
}
```

## 미로찾기

### 기본 진입

확인 항목:

- 홈에서 `미로찾기` 클릭 시 `maze-view`가 active 된다.
- 미로판이 보인다.
- 캐릭터가 왼쪽 바깥 입구에 있다.
- 출구가 바깥 벽에 연결되어 있다.

### 난이도

확인 항목:

- `쉬움`, `보통`, `어려움`, `매우 어려움` 버튼이 있다.
- 각 버튼을 누르면 새 미로가 생성된다.
- 현재 난이도 표시가 바뀐다.
- 힌트 수가 난이도에 맞게 바뀐다.

예상 힌트 수:

- 쉬움: 4
- 보통: 3
- 어려움: 2
- 매우 어려움: 1

### 이동

확인 항목:

- 키보드 방향키로 이동된다.
- 화면 방향 버튼으로 이동된다.
- 벽으로 이동하려고 하면 이동하지 않고 메시지가 나온다.
- 이동할 때 걸음 수가 증가한다.

### 힌트

확인 항목:

- 힌트 버튼을 누르면 노란색 힌트 칸이 표시된다.
- 힌트 수가 1 줄어든다.
- 힌트가 0이면 더 이상 표시되지 않는다.

### 탈출

확인 항목:

- 출구에 도착하면 탈출 성공 메시지가 나온다.
- 탈출 횟수가 증가한다.
- 성공 후에는 더 이상 움직이지 않는다.
- `새 미로`를 누르면 다시 시작된다.

### 구조 검사

로직 검사 기준:

- 시작점에서 출구까지 경로가 존재한다.
- 미로 그래프에 순환이 없어야 한다.
- `edges = nodes - 1` 조건이 성립해야 한다.

## 사진/동영상 퍼즐

### 사진 선택

확인 항목:

- `사진/동영상` 파일 입력이 있다.
- 모바일에서 누르면 갤러리/파일 선택 UI가 뜬다.
- 사진 선택 후 “사진이 선택됐어요”류의 피드백이 나온다.
- 줄/칸을 정하고 `퍼즐 만들기`를 누르면 조각이 생긴다.

### 동영상 선택

확인 항목:

- 동영상 파일을 선택할 수 있다.
- 선택 후 “첫 장면으로 퍼즐을 준비” 메시지가 나온다.
- 첫 프레임 캡처가 성공하면 퍼즐을 만들 수 있다.
- 캡처 실패 시 다른 영상을 고르라는 메시지가 나온다.

주의:

- 일부 모바일 브라우저나 코덱에서는 첫 프레임 캡처가 실패할 수 있다.
- 자동 재생은 모바일 정책상 막힐 수 있다.

### 줄/칸 입력

확인 항목:

- 최소값은 2다.
- 최대값은 8이다.
- 잘못된 값은 2~8 범위로 보정된다.
- `2 x 5`, `5 x 2` 같은 비정사각 퍼즐도 조각 비율이 크게 깨지지 않는다.

### PC 조작

확인 항목:

- 조각을 드래그할 수 있다.
- 퍼즐판 칸에 드롭할 수 있다.
- 잘못 놓은 조각도 이동할 수 있다.
- 이미 조각이 있는 칸에 다른 조각을 놓으면 기존 조각이 트레이로 돌아간다.

### 모바일 조작

확인 항목:

- 조각을 터치하면 선택된다.
- 선택된 조각에 테두리가 보인다.
- 퍼즐판 칸을 터치하면 조각이 이동한다.
- 같은 조각을 다시 터치하면 선택이 해제된다.
- 힌트로 잠긴 조각은 선택되지 않는다.

### 힌트

확인 항목:

- 힌트 버튼을 누르면 아직 맞지 않은 조각 하나가 정답 위치로 간다.
- 힌트 조각은 초록색 잠금 표시가 생긴다.
- 힌트 조각은 다시 이동할 수 없다.
- 점수가 증가한다.

### 완성

사진 퍼즐:

- 모든 조각을 맞추면 완성 메시지가 나온다.

동영상 퍼즐:

- 모든 조각을 맞추면 비디오 플레이어가 보인다.
- 자동 재생을 시도한다.
- 자동 재생이 막히면 재생 버튼을 누르라는 메시지가 나온다.
- 사용자가 재생 버튼을 누르면 동영상이 재생된다.
