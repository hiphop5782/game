## 앱 성격

아이들이 사용할 수 있는 미니게임 모음 웹앱입니다. 별도 빌드 과정 없이 `index.html`을 브라우저에서 열어 실행하는 정적 웹페이지 구조입니다.

## 주요 파일

- `index.html`: 현재 가장 중요한 파일입니다. 홈 화면, 게임 화면, 스타일, 게임 로직이 대부분 이 파일에 들어 있습니다.
- `images/`: 국기 힌트 이미지, 동물 이미지 등이 들어 있습니다.
- `css/style.css`: 초기 스타일 파일로 보이나, 현재 최신 화면 스타일은 `index.html` 내부 `<style>`에 더 많이 반영되어 있습니다.
- `js/*.js`: 초기 분리형 게임 로직 파일로 보이나, 현재 최신 통합 구현은 `index.html` 내부 `<script>`에 있습니다.

## 화면 전환 구조

`index.html` 내부의 `app` 객체가 화면 전환을 담당합니다.

- `views` 배열에 화면 id를 등록합니다.
- `startGame(type)`에서 게임 타입별로 화면을 보여주고 해당 게임의 `init()`을 호출합니다.
- 새 게임을 추가할 때는 홈 카드, `<section id="...-view">`, `views` 배열, `startGame()` 분기, 게임 객체를 함께 추가해야 합니다.
